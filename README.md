# Blogpaper

> A newspaper like blog theme for Hugo.

## Install

Get the theme:

```bash
git submodule add https://github.com/NormandErwan/blogpaper themes/blogpaper
```

Change you `config.toml`:

```toml
theme = "blogpaper"
```

## Upgrade

Execute:

```bash
git submodule update --remote themes/blogpaper
```
